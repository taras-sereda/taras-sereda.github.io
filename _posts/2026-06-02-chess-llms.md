I gave LLM $15 each and set them in a chess tournament.
LLMs are helpful for the tasks they are explicitly teached on. There is no evidence that frontier labs are training models playing chess, so I decided to set up a torunament where 2 frontier models are playing chess against each other. gpt-5.5 vs claude-4.8 with high reasoning efforts.

TLDR; gpt-5.5 beats claude-4.8 is a short game that consists of 39 moves. at the same time gpt-5.5 is a more expensive player that spent $8 while claude only $5. 
Every game starts with classic opening `e2e4` I didn't see LLMs being more experimental then that.
